# 챕터2에 대해서 정리해보자
1.변수란?

단 하나의 값을 저장할 수 있는 메모리 공간.

2.변수를 선언하는 방식

int variable; //    [데이터타입] [변수 이름];

int variable2 = 3; //   [데이터타입] [변수 이름] = [값];

첫 번째 방식은 변수를 선언만 하는 방식이고,

두 번째 방식은 변수를 선언과 동시에 초기화 하는 방식이다.

상황에 따라 선언만 할 수 있고, 초기화를 같이하실 수 있다.

3.변수의 데이터 타입

                     

boolean         논리 타입 (1비트)         boolean t = true; boolean f = false;

char            문자 타입 (2바이트)        char c = 'A';

byte            정수 타입 (1바이트)        byte b = 0x70; (범위 : -128~127)

short           정수 타입 (2바이트)        short s = 100; (범위 : -32768 ~ 32767)

int             정수 타입 (4바이트)        int i = 100; (범위 : -2^31 ~ 2^31 -1)

long            정수 타입 (8바이트)        long l = 100; (범위 : -2^63 ~ 2^63 - 1)

float           실수 타입 (4바이트)        float f = 3.14;

double          실수 타입 (8바이트)        double d = 3.141592;

4.상수란?

1)변수와는 다르게 한번 값을 저장하면 변경이 불가하다.

2)대입하려는 리터럴에 사용자가 이름을 붙여서 코드의 가독성을 좋게한다.


5. 리터럴이란? 

프로그램에서 직접 표현한 값

종류로는 정수, 실수, 문자, 논리, 문자열 리터럴이 있다.

1)정수 리터럴 - 10진수, 8진수, 16진수, 2진수 리터럴

정수 리터럴은 int 형으로 컴파일

long 타입 리터럴은 숫자 뒤에 L 또는 l을 붙여 표시한다.

2)실수 리터럴 - 소수점 형태나 지수 형태로 표현한 값이다.

실수 타입 리터럴은 double 타입으로 컴파일 된다.

숫자 뒤에 f(float)나 d(double)을 명시적으로 붙이기도 한다.

(float 같은경우는 f를 꼭 붙여줘야하고 double은 생략 가능하다.)

3)문자 리터럴 - 단일 인용부호( ' ' )로 문자를 표현한다.

4)문자열 리터럴 - ( " " )로 문자열을 표현한다. 

5)논리타입 리터럴 - boolean 타입 변수에 치환하거나 조건문에 이용
